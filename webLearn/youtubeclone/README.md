# YouTube Frontend Clone

**Project Status**

- This project is currently a work in progress.
The project will continue to be updated as I learn new front-end concepts and improve my development skills

---
This project was created as a personal coding challenge to improve my understanding of front-end web development. The goal was to rebuild the YouTube user interface from scratch by studying the original website and implementing the design independently.

AI tools were used only for research purposes, such as understanding concepts and finding documentation. No AI-generated code was used in the development of this project.
